# Lesson 14: Fetching Data from API - React App

A simple React application that demonstrates how to fetch and display data from a public API using `useEffect` and `useState`.

## ✨ Features

- Fetches user data from JSONPlaceholder API on component mount
- Loading state while fetching data
- Displays list of users with name and email
- Clean error handling in console
- Responsive and minimal UI

## 🛠 Technologies Used

- **React.js** (Functional Components + Hooks)
- `useState` and `useEffect`
- Fetch API (native JavaScript)
- JavaScript (ES6+)
- CSS (via App.css)
